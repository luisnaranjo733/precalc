Function composition and inherited domain and range
---------------------------------------------------

.. math::

   f(x) = 2x + 3
   
   \text{Domain: } (-\infty, +\infty)
   
   \text{Range: } (-\infty, +\infty)
   
   g(x) = -x^2 + 5`
   
   \text{Domain: } (-\infty, +\infty)

   \text{Range: } (-\infty, 13]
   
   \text{find} (f \circ g)(x):

   (f \circ g)(x) = f(g(x))

   f(-x^2 + 5) = 2(-x^2 + 5) + 3

   f(-x^2 + 5) = -2x^2 + 10 + 3

   f(-x^2 + 5) = -2x^2 + 13

   (f \circ g) = -2x^2 + 13

   \text{Domain: }(-\infty, \infty)

   \text{Range: }(-\infty, 13]
